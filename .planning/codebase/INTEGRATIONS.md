# Integrations

## Blockchain Networks
- **Sui Network (Testnet)**: Target deployment environment for the Move smart contracts (`coc_web3/Move.toml` references `framework/testnet`).

## External Assets & Contracts
- **Sui RPC Nodes**: Interfacing with the Sui network from the frontend via dApp kit.
- **OnchainInvoice**: External Move package dependency sourced from `Bermu-DAO/sui_workshop_3` repository.

## Authentication
- **Sui Wallets**: Via `@mysten/dapp-kit-react` for user authentication, connection, and transaction signing.
