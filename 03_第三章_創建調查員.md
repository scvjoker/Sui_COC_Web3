# 第三章 創建調查員

譯／月夜白雨

一切衝擊中最能使人狂亂的，無過於那些完全出乎預料、荒誕到極度不可信的衝擊了。我經歷過的任何恐怖都無法與眼前的景色、以及這幅景色的怪異和驚奇帶來的恐怖相比擬。

——H.P. 洛夫克拉夫特，《異鄉人》，玖羽譯

在克蘇魯的呼喚中，每位玩家都需要製作一張調查員。這是一群發掘深層秘密的人，這是一群搜索遺忘之地的人，這是一群用禁忌知識武裝自己頭腦，正面應對克蘇魯神話中那些毛骨悚然之物的人。

作為團隊來講，調查員們各有豐富多彩的背景和千奇百怪的職業——每一個都會給團隊帶來新的口味。他們或因為朋友之義聚在一起，或因為共同目標成為同志，但最終，會一同堅定地對抗襲來的暗湧。

本章提供創建調查員這一玩家角色的規則。在本章結尾你可以找到一些可選規則，以及一面雙頁大開的快速參考可供食用。

將自己的調查員從草稿狀態創建成功可是莫大的樂趣。規則會告訴你如何骰出你的屬性，然後選擇職業和技能，最後寫出個人經歷。在開始遊玩克蘇魯的呼喚前，玩家們和守秘人坐在一起骰出各自的屬性可以最大限度確保每個人都能融入團隊。因為在擲骰和討論中，你們的調查員將會擔負起不同的責任，選擇合理的技能和職業，使團隊技能分布趨於平衡。

## 創建你的調查員

在創建調查員的第一步之前，有的玩家就有一個創建意向在腦海中形成，而另外一些玩家則喜歡在擲骰結束後再做規劃。下面所述只是普通的創建方法，更多選擇會在章節末尾。若需要更多指引，請轉到第十章：主持遊戲。

### 步驟
1. 第一步：決定屬性
2. 第二步：決定職業
3. 第三步：決定技能並分配技能點
4. 第四步：寫出背景
5. 第五步：決定裝備

## 3.1 第一步：生成屬性

調查員有著 8 項作為基石存在的屬性。在遊戲中，每種屬性都表現了調查員的其中一面——記憶力啊協調性啊等等。這些屬性數字決定了調查員的相關才能和遊戲中的建議扮演趨向。

屬性值的決定通常隨機決定——通過使用兩枚或更多的 D6 骰子。將骰子結果相加後，再乘以 5，可以得出一個固定的百分比數。這就是你的最初屬性，會在 15% 到 90% 之間。

### 投擲屬性

在骰出屬性時千萬不要直接填角色卡，而是寫在草稿上，因為你有可能因為年齡而對屬性做出調整。

- **力量 (STR)**：投 3D6，結果乘 5，即是力量。
  力量是調查員肌肉能力的量化。力量越高，調查員就能舉起更重的東西或更強有力的抓住物體。該屬性會決定調查員在近戰中造成的傷害。力量降低為 0 時，調查員就成為一個無法離開床舖的病號。

- **體質 (CON)**：投 3D6，結果乘 5，即是體質。
  體質意味著健康、生氣和活力。毒藥和疾病會與調查員的體質屬性正面相鬥。高體質的調查員會有更多的生命值——能承受更多傷害和攻擊。嚴重的物理損傷或魔法攻擊有可能降低該屬性，而當體質降為 0 時，調查員就死咯。

- **體型 (SIZ)**：投 2D6+6，結果乘 5，即是體型。
  體型值將身高和體重整合成了一個數字。伸長脖子越過矮牆觀望，或者擠進狹窄的空間，或者判定誰的頭在蹲下時也會高出草堆一個截時，就看體型了。體型可以幫助決定生命值和傷害加值和體格。體型的減少通常意味著丟失肢體，當然這也意味著敏捷的減少。對於調查員來說，失去所有體型，應該意味著它消失了——只有上帝知道它在哪！

- **敏捷 (DEX)**：投 3D6，結果乘 5，即是敏捷。
  高敏捷的調查員更為迅捷靈敏，肉體更加柔韌。敏捷檢定可以幫助你在墜落中抓住支撐，或高速穿越敵人，或做到一些纖細的行動。敏捷降為 0 的調查員將會神經絮亂，無法完成任何物理行動。在戰鬥中，高敏捷的角色會優先行動（見第六章：戰鬥）。

- **外貌 (APP)**：投 3D6，結果乘 5，即是外貌。
  外貌統括了肉體吸引力和人格魅力。高外貌的人瀟灑而惹人喜愛，但不一定會有一副好面孔。外貌降為 0 的人恐怖而醜陋，有著令人十分厭惡的舉止，走到哪都會引發議論和震動。外貌會在社交活動中發生效用，或在試圖給某人留下好印象時有所幫助。

- **智力 (INT)**：骰 2D6+6，結果乘 5，即是智力。
  智力表示為調查員學習能力、理解能力、信息分析能力和解密能力的優劣度。智力降為 0 的調查員就會如同嬰兒般是一個流涎的傻瓜。智力決定了調查員的興趣技能點的數量（INT × 2）。智力同樣影響著靈感檢定和智力檢定（見第五章：遊戲系統）。

- **意志 (POW)**：投 3D6，結果乘 5，即是意志。
  意志正是心意的力量；意志越高，學習和抵抗魔法的資質就越高。意志降為 0 的調查員如同行屍走肉，沒有了「意念」，當然也無法使用魔法。除非特有說明，否則遊戲中意志的降低會是永久性的。理智值 (SAN) 的遊戲初始值等於角色的意志。更多信息請前往第八章：理智。意志同樣決定了角色的魔法值。與意志不同的是，魔法值可以在遊戲中消費並回復。魔法值等於意志的五分之一（見第九章：魔法）。

- **教育 (EDU)**：投 2D6+6，結果乘 5，即是教育。
  教育屬性是調查員所真正掌握的正規知識的量化，它表明了調查員在全日制學習中花費了多長時間。教育表示的是調查員保持的信息數量，而非機智應變使用信息的能力（那是智力的範疇）。教育為 0 的角色估計是新生兒或者失憶過——沒有關於世界的常識，就會顯得十分好奇而容易受騙。

### 那些數字意味著什麼？

- **力量**
  - 0：衰弱：沒法站起來甚至端起一杯茶。
  - 50：普通人水平。
  - 99：世界水平 (奧賽舉重冠軍)。人類極限。
- **體質**
  - 0：死亡。
  - 50：普通人水平。
  - 99：鐵之剛體。可以承受住最強的疼痛。人類極限。
- **體型**
  - 1：一名嬰兒 (1~12 磅)。
  - 65：普通人類體型 (中等身高和體重)(170 磅 / 75kg)。
  - 99：超大號 (330 磅 / 150kg)。
- **敏捷**
  - 0：沒有協助無法移動。
  - 50：普通人水平。
  - 99：世界級運動員。人類極限。
- **智力**
  - 0：沒有智商，無法理解周遭的世界。
  - 50：普通人水平。
  - 99：天才 (愛因斯坦、達文西、特斯拉等)。人類極限。
- **意志**
  - 0：弱者的心，沒有意志力，沒有魔法潛能。
  - 50：普通人水平。
  - 100：鋼鐵之心。人類極限。
- **教育**
  - 0：新生兒。
  - 60：高中畢業。
  - 90：博士學位，教授。
  - 99：人類極限。

### ⊙ 幸運
創建調查員時，骰 3D6，結果乘 5，即是幸運值。

### ⊙ 年齡
- **15-19 歲**：力量和體型合計減 5 點。教育減 5 點。幸運可骰 2 次取較好者。
- **20-39 歲**：教育進行 1 次增強檢定。
- **40-49 歲**：教育進行 2 次增強檢定。力體敏合計減 5 點。外貌減 5 點。
- **50-59 歲**：教育進行 3 次增強檢定。力體敏合計減 10 點。外貌減 10 點。
- **60 歲以上**：力體敏合計減 20 點以上。外貌大幅下降。

### ⊙ 其他屬性
- **傷害加值 (DB) 和體格**：將力量和體型相加，對照表 I 決定。
- **生命值**：(體質 + 體型) / 10，向下取整。
- **移動速度 (MOV)**：由力量、敏捷、體型和年齡共同決定。

## 3.2 第二步：決定職業

職業展示出調查員的生活方式，例如醫生、學生甚至是一個混混騙子。職業也反映出調查員的專精領域。

### 範例職業
- **古文物學家**：估價、歷史、圖書館使用、語言等。
- **記者**：藝術、歷史、圖書館使用、心理學等。
- **教授**：圖書館使用、其他語言、母語、心理學等。
- **醫生**：急救、醫學、心理學、科學等。

## 3.3 第三步：決定技能與分配技能點

- **本職技能點**：由教育 (EDU) 決定，分配給職業技能。
- **興趣技能點**：由智力 (INT) × 2 決定，可分配給除克蘇魯神話以外的任何技能。
- **信用評級**：決定了調查員的資金和社會地位。

## 3.4 第四步：創造背景

調查員的背景故事讓角色更加鮮活：
1. **形象描述**：外表特徵。
2. **思想與信念**：宗教、哲學或政治觀點。
3. **重要之人**：家人、朋友或導師。
4. **意義非凡之地**：故鄉、工作地點或靜思之地。
5. **寶貴之物**：紀念品、武器或收藏品。
6. **特質**：慷慨、忠誠、野心等。

## 3.5 第五步：決定裝備

根據信用評級決定生活水平（身無分文、貧窮、標準、小康、富裕、豪富）以及初始現金和資產。
